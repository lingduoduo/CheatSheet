⌘ + T / ⌘ + P - Jump to file

⌃+G - Jump to line

⌘ + ⇧ + P - Open command promp

⌃ + ⇧ + ⌘ + F - Toggle Distraction-free mode

⌘+L - Select Line

⌘+D - Select duplicates

⌘ + ⇧ + D - Duplicate lines

Alt +(Left/Right) - 进行逐词移动
Alt + Shift + ←/→ 进行逐词选择。

Cmd + Enter 在当前行下面新增一行然后跳至该行；Cmd + Shift + Enter 在当前行上面增加一行并跳至该行。
